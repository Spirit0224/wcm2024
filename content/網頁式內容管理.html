1.網際內容管理課程將網頁內容管理系統與網頁資料放在同一個倉儲中, 且讓使用者寄存在 Github 中

2.只要能夠執行 Python 的雲端或近端環境都能開啟網頁編輯程式進行改版, 而編輯完成的網頁資料則可在 Github 倉儲中透過 Pages 的全球資訊網伺服器功能呈現網站內容.

3.如何利用電腦輔助設計室的舊 Python 建立新的可攜環境?

5.設計運算：對於需要大量數學運算或模擬的任務，您可以使用 Python 來編寫算法並將其嵌入到網頁中，以便在用戶的瀏覽器中執行。
